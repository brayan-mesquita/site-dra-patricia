## 🎯 Objetivo da página

Esta é uma **landing page de destino para campanhas de Google Ads**, criada especificamente para o termo de busca **"implante dentário em porto velho"** e variações similares.

**Objetivo único:** converter visitantes em cliques no botão de WhatsApp para agendamento de avaliação gratuita.

Não é um site institucional. Não tem menu de navegação com múltiplas saídas. Cada elemento da página existe para conduzir o visitante a **uma única ação: clicar no WhatsApp**.

## 📱 Foco mobile

A página foi construída **mobile-first** com `max-width: 480px`. No desktop ela centraliza o conteúdo. Todas as fontes, espaçamentos e botões são calibrados para telas de celular.

**Elementos críticos no mobile:**
- Botão WhatsApp visível **sem rolar** (acima do fold no hero)
- Barra sticky verde no rodapé — presente em 100% do scroll
- Texto legível sem zoom (mínimo 14px)
- Imagens com `object-fit: cover` e `aspect-ratio` definido
